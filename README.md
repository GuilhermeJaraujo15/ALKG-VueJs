O ALKG, é um Sistema que auxilia o usuário no gerenciamento de Tarefas Empresariais. Este utiliza o Framework VueJs, 
e conta com áreas como Dashboard, para uma visão geral das atividades adicionadas, uma Agenda para reservas de compromissos,
Clientes para um melhor relacionamentos dos dados dos envolvidos na ocupação e, Estatísticas, para uma compreensão de como está indo
as atividades adicionadas.
